HOW TO DEPLOY A SPECIFIC VERSION:

On your production server, run:

# Deploy Latest (v1.1 - Time Slot Fix)

# To go back to v1.0 (if needed)
git checkout bc281be
docker exec -it pharmacy_php php artisan cache:clear
docker restart pharmacy_php

# To return to latest
git checkout master
git pull origin master
docker exec -it pharmacy_php php artisan cache:clear
docker restart pharmacy_php
